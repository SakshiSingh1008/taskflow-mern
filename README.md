TaskFlow MERN

A full-stack task management application built using the MERN stack. It helps users create, track, update, and manage tasks efficiently with authentication and a clean UI.

Tech Stack
Frontend:
React.js
Axios
React Router
CSS / Tailwind (if used)
Backend:
Node.js
Express.js
MongoDB
Mongoose
JWT Authentication
 

Features
 User authentication (Login / Register)
 Create tasks
 Update tasks
 Delete tasks
 Task filtering (All / Todo / In Progress / Done)
 User-specific task management
 Fast API integration
Clean and responsive UI


Project Structure
taskflow-mern/
│
├── client/          # React frontend
├── server/          # Node + Express backend
├── models/          # MongoDB models
├── routes/          # API routes
├── controllers/     # Business logic
└── README.md

 
 
 Installation & Setup
1. Clone repo
git clone https://github.com/SakshiSingh1008/taskflow-mern.git
cd taskflow-mern
2. Setup backend
cd server
npm install
npm start
3. Setup frontend
cd client
npm install
npm start
